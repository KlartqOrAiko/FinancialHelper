# 💰 Financial Goal Calculator

A simple C# console application that calculates how many months you need to reach a financial savings goal.

---

## 🚀 Features

- Calculates months needed to reach a savings goal
- Fractional months are always rounded up
- Input validation — no crashes on invalid values

---

## 📊 How it works

| Input | Description |
|-------|-------------|
| Savings goal | The total amount you want to save |
| Monthly savings | How much you can save per month |

The app divides your goal by your monthly savings and rounds up to the nearest whole month.

---

## ⚙️ Requirements

- [.NET 8 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) — required to build and run the app

---

## ▶️ How to Run

**1. Clone the repository**
```bash
git clone https://github.com/your-username/financial-goal-calculator.git
cd financial-goal-calculator
```

**2. Run the application**
```bash
dotnet run
```
